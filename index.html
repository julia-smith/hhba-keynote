Two types of animation:

  • Enhance the user interface
  • Enhance the data

Enhancing the UI

  • Create fluid transitions
  • Provide visual feedback
  • Guide the user

Enhancing the data

  • Illustrate change
  • Enhance relationships
  • Provide context
  • Create an emotional response
